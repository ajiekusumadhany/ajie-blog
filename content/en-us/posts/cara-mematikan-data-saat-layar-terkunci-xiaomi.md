Pernahkah Anda mengalami situasi di mana penggunaan data pada perangkat Xiaomi Anda terus berjalan meskipun layar terkunci?

Hal ini mungkin terjadi karena beberapa aplikasi atau pengaturan yang memungkinkan akses data saat perangkat dalam kondisi terkunci.

Namun, Anda tidak perlu khawatir, karena dalam artikel ini kami akan membahas cara mematikan data saat layar terkunci pada perangkat Xiaomi dengan mudah dan praktis.

Cara Mematikan Data Saat Layar Terkunci Xiaomi
----------------------------------------------

Berikut ini adalah beberapa cara yang mungkin bisa anda coba:

### 1\. Menggunakan Fitur "Background Data Restriction"

Salah satu cara termudah untuk mematikan akses data saat layar terkunci pada perangkat Xiaomi adalah dengan menggunakan fitur "Background Data Restriction" yang disediakan oleh sistem MIUI.

Fitur ini memungkinkan Anda mengatur batasan akses data untuk aplikasi tertentu saat perangkat dalam kondisi terkunci.

Berikut adalah langkah-langkahnya:

*   Buka aplikasi "Settings" (Pengaturan) pada perangkat Xiaomi Anda.
*   Gulir ke bawah dan temukan opsi "Password & security" (Kata Sandi & Keamanan).
*   Pilih "Privacy" (Privasi) dan cari opsi "Background data restriction" (Batasan data latar belakang).
*   Aktifkan opsi tersebut dan pilih aplikasi yang ingin Anda batasi akses datanya saat layar terkunci.

Dengan mengaktifkan fitur ini dan membatasi akses data untuk aplikasi tertentu, Anda dapat menghemat penggunaan data yang tidak perlu saat perangkat dalam keadaan terkunci.

### 2\. Menggunakan Mode "Data Saver"

Selain fitur "Background Data Restriction", Xiaomi juga menyediakan mode "Data Saver" yang dapat membantu Anda mengurangi penggunaan data saat layar terkunci.

Mode ini secara otomatis membatasi akses data untuk aplikasi latar belakang saat perangkat dalam kondisi terkunci.

Berikut adalah langkah-langkah untuk mengaktifkan mode "Data Saver":

*   Buka aplikasi Pengaturan pada perangkat Xiaomi Anda.
*   Scrol ke bawah dan temukan opsi "Data Usage" (Penggunaan Data).
*   Pilih "Data Saver" dan aktifkan opsi tersebut.

Dengan mengaktifkan mode "Data Saver", perangkat Xiaomi Anda akan secara efektif membatasi akses data untuk aplikasi saat layar terkunci, sehingga membantu mengurangi penggunaan data yang tidak perlu.

### 3\. Mematikan Akses Data untuk Aplikasi Tertentu

Selain menggunakan fitur sistem, Anda juga dapat mematikan akses data secara individual untuk aplikasi tertentu pada perangkat Xiaomi Anda.

Jika ada aplikasi yang tidak perlu mengakses data saat layar terkunci, Anda dapat mengatur pengaturan aplikasi secara manual.

Berikut adalah langkah-langkahnya:

*   Buka aplikasi Pengaturan pada perangkat Xiaomi Anda.
*   Temukan dan pilih opsi "Apps" (Aplikasi) atau "Applications" (Aplikasi) di menu Pengaturan.
*   Cari aplikasi yang ingin Anda batasi akses datanya saat layar terkunci dan pilih aplikasi tersebut.
*   \-Pilih opsi "Mobile data" (Data seluler) atau "Data usage" (Penggunaan data) pada halaman aplikasi.
*   \-Matikan opsi "Allow background data" (Izinkan data latar belakang) atau "Restrict background data" (Batasan data latar belakang) untuk mematikan akses data saat layar terkunci.

Dengan mematikan akses data untuk aplikasi tertentu, Anda dapat mengontrol penggunaan data saat perangkat Xiaomi dalam keadaan terkunci dan menghindari penggunaan data yang tidak perlu.

### 4\. Menggunakan Aplikasi Pengelola Data

Selain pengaturan bawaan yang disediakan oleh perangkat Xiaomi, Anda juga dapat menggunakan aplikasi pengelola data pihak ketiga untuk membantu mematikan akses data saat layar terkunci.

Beberapa aplikasi pengelola data populer, seperti "Datally" atau "GlassWire", menawarkan fitur yang memungkinkan Anda mengontrol penggunaan data dengan lebih detail.

Anda dapat mengunduh aplikasi pengelola data ini melalui Play Store atau Mi App Store.

Setelah menginstal aplikasi pengelola data yang dipilih, Anda dapat mengikuti petunjuk yang diberikan oleh aplikasi untuk mematikan akses data saat layar terkunci dan mengatur batasan penggunaan data untuk aplikasi tertentu.

### 5\. Mengaktifkan Mode Pesawat saat Tidak Digunakan

Langkah terakhir yang dapat Anda lakukan untuk mematikan akses data saat layar terkunci pada perangkat Xiaomi adalah dengan mengaktifkan mode pesawat saat perangkat tidak digunakan.

Mode pesawat mematikan semua konektivitas jaringan, termasuk akses data, sehingga memastikan tidak ada penggunaan data yang tidak perlu saat perangkat dalam keadaan terkunci.

Anda dapat mengaktifkan mode pesawat dengan cepat melalui menu notifikasi atau melalui "Settings" (Pengaturan) pada perangkat Xiaomi Anda.

Pastikan untuk menonaktifkan mode pesawat saat Anda ingin menggunakan kembali akses data dan konektivitas jaringan.

Kesimpulan
----------

Dalam artikel ini, saya telah membahas cara mematikan data saat layar terkunci pada perangkat Xiaomi dengan mudah dan praktis.

Dengan menggunakan fitur bawaan seperti "Background Data Restriction" dan "Data Saver", mematikan akses data untuk aplikasi tertentu, atau menggunakan aplikasi pengelola data pihak ketiga, Anda dapat mengontrol penggunaan data yang tidak perlu dan menghemat kuota data Anda.

Selain itu, mengaktifkan mode pesawat saat perangkat tidak digunakan juga dapat menjadi solusi yang efektif untuk mematikan akses data secara keseluruhan.

Dengan menerapkan langkah-langkah ini, Anda dapat mengoptimalkan penggunaan data Anda dan menghindari penggunaan data yang tidak diinginkan saat perangkat [Xiaomi](https://www.mi.co.id/id) Anda dalam kondisi terkunci.
